
# 📊 AI-Investment-Assistant

Интеллектуальная веб-система для анализа акций компаний на основе рыночных данных, новостей и искусственного интеллекта.

---

# 🎯 Цель проекта

Разработать веб-приложение, которое позволяет пользователю:

- получать данные об акциях компаний по тикеру;
- просматривать краткую информацию о компании;
- анализировать последние новости;
- получать AI-аналитику на основе данных и новостей.

---

# ⚙️ Функциональность

## 1. Поиск акции

Пользователь вводит тикер компании (например: `AAPL`, `TSLA`, `MSFT`).

Система должна:
- проверить корректность тикера;
- получить данные об акции;
- вернуть результат пользователю.

---

## 2. Информация об акции

Отображается:

- название компании;
- тикер;
- текущая цена;
- изменение цены за день (%);
- валюта.

---

## 3. Информация о компании

Краткое описание:

- сфера деятельности;
- основные продукты/услуги;
- позиция на рынке.

---

## 4. Новости компании

Система отображает последние новости:

Для каждой новости:
- заголовок;
- дата публикации;
- краткое содержание.

---

## 5. Анализ новостей

Для каждой новости система определяет:

- влияние на компанию (положительное / отрицательное / нейтральное);
- краткое объяснение причины.

---

## 6. AI-анализ

На основе данных и новостей ИИ формирует итоговый отчет:

Содержит:
- общую оценку ситуации;
- ключевые факторы влияния;
- риски;
- итоговый вывод.

Объём: 100–200 слов.
